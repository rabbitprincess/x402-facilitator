# x402-facilitator

**x402-facilitator** is a Go-based middleware that settles on-chain payments authorized via the [x402 protocol](https://x402.dev).

It Supports facilitator and client roles on EVM-compatible chains.


## Prerequisites
- Docker
- Docker Compose

# How to run
```bash
docker compose up
```
